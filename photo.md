## üìê Technical Specifications

### Image Dimensions
- **Backgrounds**: 1920x1080 minimum (16:9 aspect ratio)
- **Character Portraits**: 800x1200 (portrait orientation)
- **UI Elements**: Various, but design for 4K support
- **Special Effects**: 1920x1080, with alpha channel transparency

### File Formats
- **Backgrounds**: PNG or WebP with optimization
- **Characters**: PNG with transparent backgrounds
- **Effects**: PNG with alpha channel
- **UI Icons**: SVG preferred for scalability

### Style Guidelines
- **Art Style**: Semi-realistic to stylized painterly
- **Consistency**: Maintain cohesive visual language across all assets
- **Accessibility**: High contrast text-to-background ratios
- **Mood Lighting**: Use lighting to reinforce emotional tone
- **Cultural Aesthetics**: 
  - Institute: Gothic architecture, geometric precision
  - Clans: Organic, Art Nouveau curves, natural forms
  - Echoborn: Crystalline, abstract, ethereal

## üìù Notes for Artists

### Important Considerations
- **Consistency**: All assets should feel like they belong to the same world
- **Readability**: Text must be readable against all backgrounds
- **Emotional Impact**: Visuals should reinforce the narrative mood
- **Cultural Sensitivity**: The game draws on multiple cultural linguistic traditions
- **Accessibility**: Consider colorblind-friendly palettes, high contrast options

### Iterative Development
1. Start with rough sketches/mockups
2. Get feedback on composition and mood
3. Create color studies
4. Develop final rendered assets
5. Test in-game with dialogue and UI
6. Iterate based on playtest feedback

### File Naming Convention
```
location_[name]_[variant].png
character_[name]_[emotion].png
effect_[type]_[description].png
ui_[element]_[state].png
```

**Naming Rules**:
- Use lowercase for all file names
- Use underscores (_) to separate words and components, not spaces
- Multi-word components: use underscores (e.g., `whispering_woods` not `whisperingwoods`)
- Avoid special characters except underscores
- For numbered variants, use two digits with leading zero (e.g., `_01`, `_02`)
- Keep names descriptive but concise (max 50 characters)

Examples:
- `location_ravengard_main.png` - Main background for Ravengard
- `location_whispering_woods_clearing.png` - Specific area variant
- `character_lyra_neutral.png` - Lyra's neutral expression
- `character_kaelen_dravik_angry.png` - Multi-word name with emotion
- `effect_drift_zone_swirl.png` - Drift zone effect with description
- `effect_memory_fragment_01.png` - Numbered variants
- `ui_institute_icon.png` - Faction UI element
- `ui_language_badge_verdant.png` - Specific language badge
